This is the sum of all fears

The Sum of All Fears project is a program that demonstrates whether or not two numbers within an array of numbers add up to the number a user enters. For example, if the user enters the number 62 the program will search the array of numbers and identify whether a 30 and 32, or 20 and 42, or 13 and 49, etc. are present in the array. The program will let the user know if a pair of numbers were found to add up to the number entered.
